CySabioRK-v0.1 (alpha) - First release for Cytoscape 2.8 providing proof of principle

 

Introduction

We present CySabioRK an open-source Cytoscape plugin for loading reaction kinetics networks via web services from Sabio-RK.
CySabioRK is available for download at http://www.charite.de/sysbio/people/koenig/software/cysabiork/release/CySabioRK-v0.1.jar.
CySabioRK-v0.1 supports Cytoscape 2.8 and integrates with CySBML.

Current features are

  • Import of reaction kinetics Sabio-RK via RESTful web services
  • Integration with CySBML

Please try and report any problems you encounter using CySabioRK to matthias.koenig[AT]charite.de

Thanks and have fun
The CySabioRK team.

CySabioRK was developed by Matthias König and was funded by the Virtual Liver Network.

Installation

[top]

Cytoscape 2.8.2 or later has to be installed prior to the plugin installation.

[1] CySabioRK depends on CySBML which has to be installed prior to CySabioRK and provides the necessary SBML support and parsing for Cytoscape.

[2] Download the latest CySabioRK version at http://www.charite.de/sysbio/people/koenig/software/cysabiork/release/CySabioRK-v0.1.jar.

[2] Move the CySabioRK-vX.XX.jar into the Cytoscape plugin directory located in the Cytoscape installation directory under plugins
$CYTO_INSTALL/plugins/.
On Windows Systems Program Files/Cytoscape_v2.8.2/plugins/ on Mac /Applications/Cytoscape_v2.8.2/plugins.

The plugin is loaded and installed during the next Cytoscape startup.
To update to current CySabioRK version replace the old version of CySabioRK-vX.X.jar with the new version, to uninstall remove the CySabioRK-vX.X.jar from the plugin folder.
